# Terminal Auto Navigator 🤖🏁

A lightweight, fun terminal-based simulation of an autonomous vehicle navigating a grid. This is my very first GitHub project!

## Features
* Clean terminal UI using standard Python libraries.
* Frame-by-frame animation of a robot moving towards its destination.
* Zero external dependencies required.

## How to Run
Make sure you have Python installed, open your terminal, and run:

```bash
python navigator.py
